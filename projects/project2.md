# "Быки и коровы"
Приложение-игра, где один игрок(наше приложение) загадывает последовательность из неповторяющихся целых цифр длинной 4.
Другой игрок (пользователь) пытается угадать последовательность, приводя свои числа длинной 4. Загадавший(наше приложение), говорит оппоненту, 
сколько быков и коров в его числе.

**Быки** - цифры, стоящие на своих местах

**Коровы** - цифры, которые есть, но стоят не на своём месте

### Условия
1) Программа генерирует число
2) Пользователь пытается угадать (без ограничений на попытки)
3) Программа проверяет *корректность ввода** от пользователя и даёт подсказки
4) Если пользователь угадывает, сообщаем ему об этом и заканчиваем программу
5) User friendly interface!

*Корректным* считается ввод состоящий только из четырёх неповторяющихся цифр
