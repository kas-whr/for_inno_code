# Угадайка

### Задача
Создать приложение, которое генерирует число в диапазоне от 0 до 100, а пользователь пытается его угадать.

### Условия
- В приложении должно быть общение с пользователем: 
  - приветствие
  - предложение угадать число
  - подсказки больше или меньше загаданное число, чем введенное
  - если пользователь угадал, сообщить ему об этом, поздравить с победой
- Используйте все пройденные темы: бесконечные циклы, условные операторы, ввод-вывод данных, сравнение чисел

### Подсказка
Для генерации рандомного числа используйте `random`
```Python
import random # в самом верху файла

number = random.randint(0, 100)
```

### Усложнения
1) Добавить количество попыток, вести их счёт и сообщать игроку
2) Добавить возможность начинать игру заново (в рамках одного запуска программы)


### Пример работы

```commandline
Добро пожаловать в игру "угадай число"
Число загадано
Введите ваше число: 5
Загаданное число меньше, попробуйте ещё раз
Введите ваше число: 2
Загаданное число больше, попробуйте ещё раз
Введите ваше число: 3
Вы отгадали число, поздравляем с победой!
```
